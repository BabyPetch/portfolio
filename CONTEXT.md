# Portfolio Project Context

## Project Overview

This project is a long-term personal portfolio and developer hub for Attawat Kammas.

The website is intended to evolve continuously throughout my software engineering career and serve as:

- Personal portfolio
- Technical showcase
- Project archive
- Blog platform
- Developer identity website

The project focuses on clean architecture, scalability, maintainability, and professional presentation.

---

# Primary Goals

## Main Objectives

- Showcase Full Stack and AI/ML projects
- Present technical skills professionally
- Demonstrate real-world engineering thinking
- Maintain long-term scalability
- Support future content expansion (blogs, case studies, experience, etc.)
- Build a strong developer identity

# Project Philosophy

This project should prioritize:

- Clean UI
- Strong readability
- Minimal but modern design
- Technical professionalism
- Responsive design
- Reusable components
- Scalable architecture

The portfolio should feel like a professional engineering product rather than a beginner tutorial project.

Avoid:
- Overly flashy animations
- Excessive visual effects
- Heavy 3D rendering
- Unnecessary complexity
- Fake skill percentages

---

# Design Direction

## Visual Style

- Minimal
- Dark modern theme
- Clean typography
- Spacious layout
- Technical aesthetic

## UI Principles

- Projects are the main focus
- Content clarity is more important than animation
- Mobile responsiveness is critical
- Layout consistency matters
- Use subtle motion only where useful
