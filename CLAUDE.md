# Claude Project Instructions

## Project Overview

This repository contains a long-term personal portfolio and developer hub for Attawat Kammas.

The project represents professional growth as a Full Stack Developer and AI enthusiast.

This is NOT a tutorial-style portfolio project.

The website should feel:
- modern
- technical
- scalable
- maintainable
- production-oriented

---

# Primary Goals

The portfolio should:

- showcase real engineering projects
- highlight Full Stack and AI/ML skills
- demonstrate technical thinking
- prioritize project presentation
- remain maintainable long-term

Future expansion may include:
- technical blogs
- case studies
- engineering writeups
- architecture breakdowns

---

# Tech Stack

## Frontend
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Deployment
- Vercel

---

# Development Philosophy

Prioritize:
- readability
- maintainability
- scalability
- reusable architecture
- responsive design
- clean UI

Avoid:
- overengineering
- unnecessary abstractions
- overly complex state management
- excessive animations
- flashy beginner portfolio patterns

---

# Design Direction

The design should feel:

- minimal
- dark modern
- technical
- clean
- project-focused

The portfolio should resemble:
- modern product engineer portfolios
- startup engineering websites
- clean developer showcase platforms

Avoid:
- gaming aesthetics
- neon cyberpunk overload
- particle backgrounds
- excessive motion effects
- heavy 3D effects
