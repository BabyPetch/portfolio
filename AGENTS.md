# AI Agent Rules

## General

- Keep code modular
- Prefer reusable components
- Avoid duplicate UI
- Prioritize readability
- Keep architecture scalable

---

## Components

- Large sections go into /sections
- Reusable UI goes into /components
- Avoid massive page.tsx files

---

## Styling

- Use Tailwind CSS
- Keep spacing consistent
- Prioritize responsive design
- Avoid inline styles

---

## Data

- Keep project data in /data
- Avoid hardcoded content inside UI
- Prefer mapping data structures

---

## TypeScript

- Use explicit types when useful
- Avoid unnecessary any
- Create shared interfaces in /types

---

## Animation

- Use subtle Framer Motion only
- Avoid excessive motion
- Do not use heavy visual effects

---

## Project Philosophy

This portfolio should feel:
- professional
- minimal
- technical
- scalable

Avoid:
- tutorial-style architecture
- flashy beginner effects
- overengineering
